## 🎯 The problem

Restaurant managers waste hours switching between disconnected tools.
AvoBurger brings sales analytics, orders, products, drivers and locations
into a single, clean interface — every screen answers a real operational question.

## ✨ Features

- 📊 **Analytics dashboard** — sales trends (Recharts), KPIs, live orders table
- 🍔 **Product catalog** — category filtering, search, quick-add card
- 🛵 **Driver tracking** — status badges, live delivery detail panel
- 📍 **Multi-location** management
- 🔐 **Auth flow** — protected routes + demo login

## 🚀 Run locally

```bash
git clone https://github.com/MoussaB06/avoBurger.git
cd avoBurger
npm install
npm run dev
```

Then open http://localhost:5173.
